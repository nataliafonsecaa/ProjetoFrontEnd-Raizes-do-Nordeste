## 📖 Sobre o Projeto

Desenvolvido como estudo de caso para a disciplina de **Projeto Multidisciplinar da UNINTER**, o **Raízes do Nordeste** é um sistema web focado na modernização dos canais de atendimento de uma rede de restaurantes em expansão.

A aplicação foi projetada para resolver problemas reais de negócio, simulando as seguintes funcionalidades:

* **Cardápios por Unidade:** Exibição de pratos e preços específicos para as filiais de Tupã, Marília e Presidente Prudente.
* **Fluxo de Pedidos e Pagamento:** Seleção de itens, fechamento de carrinho e simulação de pagamentos via PIX, cartão ou dinheiro.
* **Programa de Fidelidade:** Sistema integrado de acúmulo de pontos por compra e resgate de descontos.
* **Conformidade com a LGPD:** Tela de cadastro e login com controle de consentimento de dados e política de privacidade simulada.

  Tecnicamente, o projeto é focado em front-end, utilizando HTML5, CSS3 e JavaScript, com forte prioridade na responsividade e na acessibilidade para diferentes dispositivos.
